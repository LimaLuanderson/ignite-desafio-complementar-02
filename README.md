# WatchMe
Um desafio do bootcamp <strong>Ignite da Trilha React</strong> da <a href="https://rocketseat.com.br/">Rocketseat</a>

## Sobre o desafio
O principal objetivo era refatorar uma página para listagem de filmes de acordo com gênero. Para resolver o desafio era necessário
dividir a aplicação em pelo menos duas partes principais: sidebar e o conteúdo principal que possui o header e a listagem de filmes.

## Executar a aplicação
Navegue até a pasta criada, abra no Visual Studio Code e execute os seguintes comandos no terminal:

```
yarn
yarn dev
yarn server
```

## Tecnologias Utilizadas
- [React](https://pt-br.reactjs.org/)
